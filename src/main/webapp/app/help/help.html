Recommended software:

MetaMask

MetaMask is an extension for accessing Ethereum enabled distributed applications, or "dAapps" in your normal Chrome browser!

MetaMask also lets the user create and manage their own identities, so when a Dapp wants to perform a transaction and write to the blockchain, the user gets a secure interface to review the transaction, before approving or rejecting it.

Because it adds functionality to the normal browser context, MetaMask requires the permission to read and write to any webpage. You can always "view the source" of MetaMask the way you do any Chrome extension, or view the source code on Github: https://github.com/MetaMask/metamask-plugin

You can download MetaMask on the chrome web store and read more instruction on how to use it on the MetaMask website: metamask.io


Mist Browser

Mist can be used for tokens transfers, but not supported yet for Ethereum address verifications

To use smart contract function you can install Mist browser.

Mist browser is an Internet browser (like Google Chrome) with Ethereum node connected to it.

Mist browser is free and open sourced.

Installing Mist Browser

Go to github.com/ethereum/mist/releases and download Mist for your operating system.

Run Mist Browser

When Mist Browser starts, it needs to synchronize your local copy of Ethereum blockchain and start Ethereum node. For storing blockchain you will need about 100 GB of free space on your hard drive.

When Mist Browser starts first time it may need up to few hours (depends on you network and system) to download your copy of the Ethereum blockchain.

When Mist starts you can choose to use TestNet (for testing, with test ether), or MainNet (with real ether and legal binding smart contract on it). If you run Mist for the first time, than it's probably better to start with TestNet. We have our smart contracts both for TestNet where you can test their functions, and for MainNet.
